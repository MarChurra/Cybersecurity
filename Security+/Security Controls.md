- The lower categories, fit within the 4 main categories below.

## Technical Controls
- Controls implemented using systems
- Operating syste controls (policies)
- Firewalls
- Antivirus
- IPS / IDS?

## Managerial Controls
- Administrative controls associated with security design and implementation
- Security policies, standard operating procedures

## Operational Controls
- Implemented by people instead of systems, such as security guards, or cameras
- Awareness programs
- Guard Dogs

## Physical Controls
- Limit physical access
- Guard Shack
- Fences, locks
- Badge Readers
- Authentication with fingerprint or ID

## Control Types

### Preventine
- Blocks access to a resource
  - Follow security policy
  - Firewall rules
  - Guard Shack Checks all IDs
  - Door Locks

### Deterrent
- DIscourage an intrusion attempt
- Does not prevent access
  - Application Splash Screens
  - Threat of Demotion
  - Front Reception Desk
  - Posted Warning Signs
 
### Detective 
- Identify and log an intrusion attempt
- May not prevent access
  - Collect and review system logs
  - Review login reports
  - Regularly patrol
  - Enable motion detectors

### Corrective
- Applies a control after an event has been detected
- Reverse the impact of an event
- Continue operating with minimal downtime
  - Restore from backups
  - Create p+olicies for reporting security issues
  - Contact law enforcement to manage criminal activity
  - Use a fire extinguisher
 
### Compensating
- Control using other means
- Existing controls are not sufficient
- May be temporary
  - Firewall blocks a specific app instead of patching the app
  - Implement a separation of duties
  - Require simultaneous guard duties
  - Generator used after power outage

### Directive
- Direct a subject towards security compliance
- A relatively weak security control
  - Store sensitive files in a protected folder
  - Create complaince policies and procedures
  - Train users on proper security policy
  - Post a sign for "Authrorized Personnel Only"
