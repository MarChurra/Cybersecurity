### Tables, Rows and Columns
- Each column represents one type of information, such as a name or a price
- Each row repsents a complete record, and the table contains all the records.
- A database is a place where a computer stores information in an organized way.
- You can think of a database as a digital notebook that never runs out of pages, allowing one to search, count, and sort information, quickly.
- Inside a database, information is stored in **tables**
- A Table resemlbes a spreadsheet, where information is organized into rows and columns.
- Columns describe the type of information stored and rows contain one complete set of information

### SQL
- SQL is a language to ask questions to a database.
- Instead of reading the table row by row, SQL lets the computer do the work for us.
- These questions are called queries
- A query does not change data, it only displays requested information from the table.
- The * symbol means select all
- FROM tells the database which table to use
- SELECT chooses specific columns
- WHERE filters rows, keeps only those with a matching condition
- ORDER BY sorts results by a column. By default, results are stored in ascending order.
