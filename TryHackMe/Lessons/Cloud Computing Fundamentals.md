## Cloud Benefits
- Provides scalability
- On-demand self-service
- Pay only for what you use
- Security
- High availability
- Global Access

## Types of Cloud
- Public Clolud: Used by startupts, websites and global apps because its affordable, easy to scale, and requires no infrastructure management.
- Private Cloud: Used by banks, healthcare and government organizations because it offers great control, customization and compliance for sensitive data
- Hybrid Cloud: Used by companies like e-commerce platforms that need to keep sensitive data private while still scaling publicly during high demand.

## Cloud Services
- IaaS infrastructure as a Service - You rent basic computing resources such as virtual servers, storage and networking. You are responsible for managing the operating system and your application, while the provider manages the physical hardware.
- PaaS Platform as a Service - The cloud provider manages the infrastructure and the operating system. Youj focus on building, deploying and running your application without worrying about servers
- Saas Software as a Service - You use a complete application over the internet. The provider m,anages everything, and you access the software through a browser or app.
