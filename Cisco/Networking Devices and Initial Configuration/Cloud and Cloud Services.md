### Types of Clouds
- Public clouds -
  - Cloud based applications and services offered in a public clouid are made available to the general population.
  - Services may be free or are offered on a pay-per-use mode, such as paying for online storage. It uses the internet to provice services
- Private Clouds -
  - Cloud-based applications and services offered in a private cloud  are intended for a specific organization or entity, such as the government.
  - A private cloud can be set up using the private network of an organization, though this can be expensive to build and maintain.
  - Can also be managed by an outside organization with strict access security.
- Hybrid Clouds -
  - A hybrid cloud is made up of two or more clouds (for example, part private, part public), where each part remains a separate object, but both are connected using a single architecture.
  - Individuals on a hybrid cloud would be able to have degrees of access to various services based on user access rights.
- Community Clouds -
  - Created for exclusive use by a specific community. It was customized for this community, but works like a public one.
  - For example healthcare must remain compliant iwth policies and laws that require special authentication and confidentiality

### Cloud Computing and Virtualization
- Dedicated Servers -

### Advantages of Virtualization
- Less equipment is required
- Less energy is consumed
- Less space is required
- Easier ptotyping
- Faster server provisioning
- Increased server uptime
- Improved disaster recovery
- Leagcy Support

### Hypervisor
- A program, firmware or hardware that adds an abstraction layer on top of the phyisical hardware.
- It creates virtual machines which has access to all the hardware of the phyiscal machine.
- There is the Type 1 - Bare Metal, which runs directly on the hardware, used on enterprise and data center networking devices
- The Type 2 - Hosted, uses a software that creates and runs VM instances.
