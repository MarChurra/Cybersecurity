## The Cybersecurity Code
- A useful way to think about protecting data. Cybersecurity, the CIAD triad, is divided in three dimensions, of information security.

### Security Principles
- The foundational principles of confidentiality, integrity and availability of data provide a focus which enables the cybersecurity expert to prioritize actions when protecting any networked system.
- Data confidentiality prevents the disclosure of information to unauthorized people, resources, or processes.
- Data integrity refers to the accuracy, consistency, and trustworthiness of data.
- Data availability ensures that information is accessible by authorized users when needed.

### Data States
- The three possible data states:
  - Data in transit
  - Data at rest or in storage
  - Data in process
 
### Safeguards
- defines the pillars on which we need to base our cybersecurity defenses in order to protect data and infrastructure in the digital realm.
- These are technology, policy and practices, and improving education, training and awareness in people.

### CIA Triad- Confidentiality
- Confidentiality without encryption, requires tokenization, which isolates data elements from exposure to other data systems. A random value with no mathematical relationship replaces original data.
- Outisd ethe system, a token has no value and is meaningless.
- It can preserve the data format, making it useful for databases and card payment processing.
- Rights management covers both digital rights management (DRM) and information rights management (IRM). Both protect data from unauthorized access by using encryption.
- DRM protects copyrighted material like music, films, or books. When any such content appears in digital form — for instance on CD, mp3, or e-book — it is encrypted, so the media cannot be copied without the decryption key. The decryption key is available only to licensed parties.
- IRM is used with email and other files that are relevant to the activities and communications of an organization. When this information is shared with others, IRM allows the document owner, the organization, or one of its members to control and manage access to the document.

### Data Integrity 
- Data undergoes several operations, such as capture, storage, retrieval, update, and transfer. Data must remain unaltered by unauthorized entities during all these operations.
- Methods used to ensure data integrity include hashing, data validation checks, data consistency checks, and access controls. Data integrity systems can include one or more of these methods.

### Availability
- Availability refers to the need to maintain availability of information whenever it is needed. Cyberattacks and system failures can prevent access to information, systems and services.
- Some exampple sto ensure uptime are:
  - Equipment Maintenance
  - OS and software update and patches
  - Backup Testing
  - Disaster Planning
  - New Technology Implementations
  - Activity Monitoring
  - Availability Testing
 
## States of Data
### Data at Rest
- Refers to data in storage. Simply put, it is the state data is in when no user or process is accessing, requesting or amending it.
- DAS Direct Attached Storage: Storage connected to a computer.
- RAID REdudant array of independent disks: These professional storage solutions use multiple hard drives in an array, which is a method of combining multiple disks so that the operating system sees them as a single disk. RAID provides improved performance and fault tolerance.
- NAS Network Attached Storage
- Storage Area Network SAN: SAN architecture is a network-based storage system. SAN systems connect to the network using high-speed interfaces, which allows for improved performance and the ability to connect multiple servers to a centralized disk storage repository.
- Cloud Storage

### Methods of Transmitting Data
- Data in transit is the second state of data, which is being transmitted.
- Data transmission involves sending information from one device to another, and protecting data in transit poses challenges. There are numerous ways to transmit data between devices.
  - Sneaker Net: Uses removable media to physically move data from one computer to another. Organizations will never be able to fully eliminate the use of a sneaker net as a way to transmit data between devices.
  - Wired Networks: Copper and fiber optics media and can serve a LAN or span great distances in WAN.
  - Wireless Networks: Use radio waves to transmit data.
